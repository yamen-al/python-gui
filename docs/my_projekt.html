my_projekt¶

my_projekt.py

Dieses Modul stellt eine GUI-Anwendung zur Echtzeit-Visualisierung und Speicherung von seriellen Daten bereit.

Klassen:

SerielleApp: Haupt-GUI-Klasse zur Verwaltung der seriellen Kommunikation. MockSerial: Klasse zur Simulation einer seriellen Schnittstelle für Testzwecke. SerielleKom: Klasse zur Verwaltung der seriellen Kommunikation.

Hauptprogramm:

Startet die GUI-Anwendung und eine Testinstanz der seriellen Kommunikation.
